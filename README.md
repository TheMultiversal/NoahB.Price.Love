# Noah B. Price - Author Website

A professional author website for Noah B. Price, featuring his books, blog, and contact information.

## Getting Started

1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open http://localhost:3000 in your browser.

## Features

- Home page with introduction and featured book
- About Me page with personal story
- Books page with catalog and individual book details
- Blog for updates and reflections
- Contact page with email signup and social links
- Image upload functionality

## Project Structure

- `index.js`: Main server file with Express routes
- `public/`: Static files including HTML, CSS, and images
- `uploads/`: Directory for uploaded images
- `package.json`: Project configuration
